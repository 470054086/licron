# licron 定时任务管理系统
1. 支持http方式,管理定时任务 暂时只有接口的方式,未开发前端管理页面
2. 支持定时任务,常驻内存服务,一次性任务的添加和杀死,(修改和重启暂时还未开启)

# 使用方式
1. 执行cron.sql 生成sql表和测试数据
2. config-local.yaml 配置文件 更改数据库链接方式,http端口
3. go run mian.go 启动服务

# 接口说明
## 定时任务接口使用方法
1. cron/lists  获取定死任务列表
2. cron/add    添加定时任务
3. cron/kill 杀死定时任务
## 常驻内存接口使用方式
1. daemon/add  添加任务
2. daemon/kill  杀死任务
## 一次性任务接口使用方式
1. once/add  添加任务
2. once/kill  杀死任务
